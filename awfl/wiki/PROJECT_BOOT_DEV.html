2. Tabs & Features

a. DEV Mode (Main IDE)

Toolbar Features:

File Operations:

New — Clear editor, reset preview, clear cache.

Open — Open a local file via File Picker or cache fallback.

Save — Save file to device or cache if unsupported.

Copy All — Copies all code in the editor to clipboard.

Preview Controls:

Preview — Render editor content in live iframe.

Reload — Refresh preview iframe.

Auto Preview — Toggle automatic live preview.
